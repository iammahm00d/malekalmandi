# ملك المندي - Link Page

A simple static landing page for **ملك المندي** restaurant links.

## Files

- `index.html` — main page, ready for GitHub Pages.

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. GitHub will publish the site and show the live URL.

## Update links

In `index.html`, replace each `href="#"` with the real URL or phone link, for example:

- Instagram: `https://instagram.com/your-account`
- TikTok: `https://tiktok.com/@your-account`
- WhatsApp: `https://wa.me/9665XXXXXXXX`
- Call: `tel:+9665XXXXXXXX`
- Location: Google Maps URL
- Menu / Order: online menu URL
