# ملك المندي Link Page

A simple responsive HTML/CSS link page for **ملك المندي**.

## Files

- `index.html` — page structure and content
- `style.css` — page styling

## How to use

1. Replace the `href="#"` values in `index.html` with the real Instagram, TikTok, location, WhatsApp, call, and menu links.
2. Upload the files to a GitHub repository.
3. Enable GitHub Pages from **Settings → Pages**.
4. Select the branch and root folder, then save.

## Local preview

Open `index.html` in your browser.
